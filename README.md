# Cybersecurity Practices and Laboratories

It is my practice in hands-on cybersecurity activities and notes on lab work stored in this repository, mainly, on the basis of TryHackMe learning paths. This repository is aimed at recording my practiced, tools utilized, and the most important concepts that I have acquired in the process of establishing a solid ground on cybersecurity.

Instead of concentrating on a completion of rooms, this repository pays attention to learning, procedure, and a lesson learned after each laboratory. This is all personally learned as I write and influenced by my approach to every topic as I practiced.

---

## Purpose of This Repository

The primary aims of having this repository are:

- To maintain organized records of cybersecurity ideas that have been acquired in real labs.  
- To record actual command usage, tools and work flows.  
- To support learning by written instructions.  
- To illustrate practical application and conformity in the training of cybersecurity.  

The repository will additionally be a personal reference in case of revision and further learning.
